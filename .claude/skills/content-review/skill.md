# Content Review Skill

Checks text completeness, missing sections, and content flow.

## Trigger
Run `/content-review` when:
- A section feels incomplete
- User mentions missing text or gaps
- After major restructuring
- Before final submission

## Review Checklist

### Per Chapter
1. **Title and intro**: Does section have clear purpose statement?
2. **Content completeness**: Are all promised topics covered?
3. **Image support**: Do images have context (not orphaned galleries)?
4. **Flow**: Does text lead reader through the section?
5. **German quality**: Is language natural, not stilted?

## Content Standards

### Travel/Personal Sections (01-russland-reise.md)
- First-person narrative ("Ich stand am Fenster...")
- Evocative but concise
- Connect visual to feeling
- Diary-like authenticity

### Reference Sections (03-architektur-vibes.md)
- Minimal text, images speak
- Brief context only
- Category headings if helpful

### World-Building (04-07)
- Clear, direct descriptions
- Specific details, not vague
- Consistent voice

## After Review
1. List issues found
2. Prioritize by impact
3. Fix or flag for discussion
4. Update chapter status table
